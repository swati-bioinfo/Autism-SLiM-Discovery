- [x] **7.8 — Method replicate: SHARK-capture (skipped)**
  **Decision:** Not used. Rationale documented in `04-docs/parameters.log`.
  **Reasons:**
  1. **No discriminative mode** — SHARK-capture finds conserved k-mers across
     orthologous sequences, not differential motifs between case/control sets.
     Cannot use our length-matched control IDRs as a negative.
  2. **Positive control passed** — MEME discriminative mode successfully detected
     all 4 spiked SLiM classes (SH3 Class I/II, 14-3-3, PDZ) at E<0.05,
     validating the pipeline as-is.
  3. **No public web server** — Only local CLI (`pip install bio-shark`);
     SHARK-capture.biocomp.unibo.it did not resolve.
  4. **Benchmark context** — SHARK-capture outperforms MEME on *standard* mode,
     but our discriminative MEME with PSP + length-matched controls is a
     different and more appropriate design for case/control discovery.
  **Alternative considered:** STREME (MEME Suite, different algorithm) — not
  needed since positive control passed.

## Phase 7: Statistical Analysis

### Step 9 — Enrichment Testing

- [x] **9.1 — Build 2×2 contingency tables**
  For each motif:
  | | Autism IDRs | Control IDRs |
  |---|---|---|
  | Motif present | a | b |
  | Motif absent | c | d |

- [x] **9.2 — Run Fisher's exact test**
  One-sided (enrichment in autism > control).

- [x] **9.3 — Apply FDR correction (Benjamini-Hochberg)**
  Threshold: q < 0.05.

- [x] **9.4 — Calculate effect sizes**
  Script: `scripts/09-enrichment-test.py`
  - Reads MEME XML motif PWMs, scans all 1502 autism + 1130 control IDRs
  - Uses each motif's `bayes_threshold` to classify presence/absence
  - Builds 2×2 tables, one-sided Fisher's exact test, BH correction
  - **Results (q < 0.05):**
    - **Motif 2 (poly-Q)** — 41/1502 autism (2.7%) vs 7/1130 control (0.6%), OR=4.5, fold=4.4, **q=0.00022**
    - **Motif 1 (poly-H)** — 12/1502 autism (0.8%) vs 0/1130 control (0.0%), OR=∞, **q=0.0059**
    - **Motif 3 (structured)** — 6/1502 vs 0/1130, p=0.034 but **doesn't survive FDR** (q=0.086)
    - Motifs 4-10: not significant
  - Output: `03-analysis/enrichment-results.csv`
